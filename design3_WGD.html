San Franciso Crime Predictor

Our Approach
Our approach started with reviwing various data sets from Kappgle. We selected the San Francisco Crime Dataset because the data is clear and easy to understand, large enough to support different predictions, lead to precise Data visualizations. Our source data obtained from Kaggle (https://www.kaggle.com/roshansharma/sanfranciso-crime-dataset) came in the form of a CSV file containing 13 columns and 150,500 rows of data. The data provides information such as the day, time, location, description, resolution, etc. of the crimes reported. There are three goals that we hope to answer in this assignment. First, is to use a machine learning model to predict outcomes based on the data we have. Second, we hope to answer the following questions: 1. Can we predict whether or not a person will be arrested and booked based on their crime? 2. Are we able to predict the type/category of crime based on the day, time and location of the crime? 3. Can we predict if certain areas are more lenient/strict than others when it comes to their outcomes based on the resolution and location?
Our Approach
Data Cleaning and Analysis. Pandas is the primary tool used for initial data analysis and cleaning of this data set. Pandas is a powerful library within the Python programming language that allows analyzing, organizing, sorting, filtering, pivoting, aggregating, calculating, a data set. Additional datal analysis will be executed using Python and its associated packages. Database Storage. Our team will utilize PostgreSQL to store data acquired for this project. PostgreSQL is a reliable and powerful, open source object-relational database system that is very suitable for this project. Machine Learning. A variety of machine learning models are being evaluated to evaluate the limitations and benefits of each prior to deciding. We will utilize Scikit-learn as a classifier. Dashboard Several different approaches to data display are in evaluation including initial data analysis with tools such as Matplotlib. Further visualization tools may include Plotly or Tableau. These visualizations will be used to create an interactive dashboard. While the approach to this dashboard is currently being selected, one approach to host this content is through Github Pages.
